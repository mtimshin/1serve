# Roadmap: Первая подача (1serve)

## 🎯 Архитектура контента (через @nuxt/content)
Сайт работает без БД. Весь контент хранится в папке `/content` в формате Markdown (`.md`). 
Администратор добавляет новые турниры, новости и корты простым созданием .md файлов с Frontmatter параметрами (title, date, image, category).

## 🗺 Структура разделов
- **Главная:** Сборный дашборд (сниппеты из других разделов).
- **Турниры (ATP / WTA):** Каталог турниров -> Календарь -> Страница турнира (шаблон статьи с боковой навигацией).
- **Wiki:** Правила, Покрытия, Ракетки, Кроссовки и одежда, Удары (шаблон статьи). Игроки (Каталог + фильтры).
- **Новости / Статьи:** Лента карточек -> Страница материала (шаблон статьи, без навигации для новостей).
- **Прогнозы:** Лента карточек -> Страница прогноза (шаблон статьи + плашка итога).
- **Любителям:**
  - Школы: Каталог -> Страница школы (Галерея, контакты).
  - Кэмпы: Каталог -> Страница кэмпа (шаблон статьи).
  - Корты: Продвинутый каталог (Фильтры: тип, покрытие, город, цена) -> Страница корта (Галерея, характеристики).
  - Любительские турниры: Каталог -> Страница турнира (шаблон статьи с боковой навигацией).
- **О проекте:** Текстовая страница.

## 🚀 Глобальные требования
1. **Единообразие:** Единый дизайн-код для всех списков, фильтраций и выдачи.
2. **SEO-оптимизация:** Настройка `robots.txt`, автогенерация `sitemap.xml`, микроразметка Article и Breadcrumbs.
3. **Адаптивность:** Десктоп начинается строго от 1024px.
